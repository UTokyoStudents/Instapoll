# とうのふだ v0——仕様書

## 使用するもの

```
+---------------------------------------------+
| 投票券 | [発行区分]        | 集計票           |
| 投票コード  集計コード(控)  |  集計コード      |
| +-------+  +--------+     | +--------+      |
| |  QR   |  |  QR    |     | |  QR    |      |
| |       |  |        |     | |        |      |
| +-------+  +--------+     | +--------+      |
+---------------------------------------------+
```

- 有権者1名に1葉送付．（発行時に自動的に破棄せられ保存せられない．）
- 切り離して回収できる様になっている．
